<h2><b>E-Commerce Sales & Order Analysis</b></h2>

<b>Project Overview</b>

This project analyzes e-commerce order data to uncover key business insights related to revenue, customer behavior, product demand, and cancellations.

The goal is to help stakeholders understand:

- Revenue performance
- Order value trends
- Quantity purchasing patterns
- Customer type differences (B2B vs B2C)
- Cancellation behavior

<b>Key Metrics Analyzed</b>
- Revenue Metrics
  - Total Revenue
  - Monthly Revenue Trend
  - Average Order Value (AOV)
  - AOV by Customer Type (B2B vs B2C)
  - Monthly AOV Trend

- Order & Quantity Metrics
  - Average Quantity per Order
  - Average Quantity per Order by Category
  - Category-wise Demand Analysis

- Cancellation Analysis
  - Overall Cancellation Rate
  - Monthly Cancellation Rate
  - Impact of Cancellation on Revenue

<b>Tools & Technologies Used</b>

- Python
- Pandas → Data cleaning & transformation
- Matplotlib → Data visualization
- Jupyter Notebook

<b>Key Insights</b>

- B2B customers generally have higher AOV compared to B2C.
- Certain categories show higher average quantity per order, indicating bundling opportunities.
- Cancellation rates fluctuate monthly, impacting revenue stability.
- Seasonal trends observed in revenue and AOV.

<b>Challenges Faced</b>

- Handling inconsistent order status values
- Managing cancellation keyword filtering
- Deciding between row-wise order count vs unique order count
- Ensuring consistency across KPI calculations

<b>Business Recommendations</b>

- Focus on high AOV customer segments (especially B2B).
- Promote bundles in high-quantity categories.
- Investigate high-cancellation months to reduce revenue leakage.
- Implement targeted campaigns during low-revenue periods.
